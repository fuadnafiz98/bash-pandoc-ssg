Installing & Configuring postgresql

Installing postgresql via postgresql apt repository

#!/bin/bash
  sudo apt update && sudo apt upgrade
  sudo sh -c 'echo "deb http://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list'
  wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -
  sudo apt update
  sudo apt install postgresql-13 postgresql-contrib

  • Enable postgresql if it doesn’t enabled already. sudo systemctl start postgresql

  • Disable postgresql if it doesn’t enabled already. sudo systemctl stop postgresql

  • Restart sudo systemctl restart postgresql
