Systemd in Linux

Random definitions

systemd is a linux initialization system and service manager that enables

  • on demand starting on daemons.
  • mount and auto-mount point maintenance
  • snapshot support
  • processes tracking using linux control groups.

It initializes the system. It has journaling system. There are alternative systems that are available to linux systems and can do the same thing systemd does.
