# COOKIES

> browser always sends the cookies.
>
> with every request.

## Creating Cookies

- with javacript - `document.cookie`
- with server - `set-cookie` header

## Cookie Properties

- make available the cookie for all the domains

  `document.cookie="all=true; domain=.example.com"`
